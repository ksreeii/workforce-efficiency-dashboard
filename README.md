# workforce-efficiency-dashboard
## This project is an interactive Excel dashboard designed to analyze workforce efficiency, productivity, and budget utilization across different divisions. The dashboard provides insights into employee performance, overtime trends, and financial efficiency using dynamic visualizations and slicers.
## Dataset Used
- <a href="https://github.com/ksreeii/workforce-efficiency-dashboard/blob/main/Production%20Working%20Hours%20Analysis%20Data%20og.xlsx">dataset
# Key Features
-KPI Tracking (Employees, Hours, Efficiency)
-Employee Performance Analysis (Overperformers vs Underperformers)
-Division-wise Work Distribution
-City-wise Overtime Earnings Analysis
-Top 5 Performing Employees
-Budget Allocation vs Actual Earnings
-Interactive Filters:
-Gender
-City
-Division
-Dashboard interaction <a href="https://github.com/ksreeii/workforce-efficiency-dashboard/blob/main/Production%20Working%20Hours%20Analysis%20Data%20og.xlsx">View dashboard</a>
# Process
-Defined problem statement and KPIs
-Cleaned and prepared dataset in Excel
-Created calculated metrics (Efficiency, Overtime)
-Built pivot tables for analysis
-Designed interactive charts and dashboard
-Added slicers for dynamic filtering
-Generated business insights and validated results
